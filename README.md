# ftw.py-tutorial

Ein Jupyter Tutorial zum Erlernen der Python Basics.

## Installation

```
git clone git@github.com:4teamwork/ftw.py-tutorial.git
cd ftw.py-tutorial
path/to/python3 -m venv .
source ./bin/activate
pip install -r requirements.txt
```

## Run Notebook

```
cd ./py-tutorial
jupyter notebook
```
